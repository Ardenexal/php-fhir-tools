# Extensions, Modifiers & Obligations

Covers validation concerns around extensions and obligations:

* **Extension context** — `type=element`, `type=fhirpath`, `type=extension`
* **Modifier extensions**
* **Obligation enforcement** — actor-scoped, via `FHIRObligationContext`
* **MustSupport reporting**
