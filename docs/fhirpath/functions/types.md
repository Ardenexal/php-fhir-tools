# Type Conversion & Checking

**Conversion:** `toBoolean()`, `toInteger()`, `toDecimal()`, `toDate()`, `toDateTime()`, `toTime()`,
`toQuantity()`, `toString()` and the matching `convertsTo*()` variants.

**Checking:** `ofType()`, `type()`, plus the `is` / `as` operators (see
[Expressions & Operators](../expressions.md)).
