Getting Started

PHP FHIRTools is a comprehensive toolkit for working with FHIR resources in PHP applications. This guide will help you get started with installation and basic usage.

Installation

Using Composer (Recommended)

composer require ardenexal/fhir-bundle

This will install the FHIRBundle which includes all components:

  • Serialization Component
  • FHIRPath Component
  • Models Component (R4, R4B, R5)
  • Symfony Integration

Standalone Components

You can also install individual components:

# Serialization only
composer require ardenexal/fhir-serialization

# FHIRPath only  
composer require ardenexal/fhir-path

# Models only
composer require ardenexal/fhir-models

Requirements

  • PHP 8.2 or higher
  • Symfony 6.4 or 7.4 (for FHIRBundle)
  • Composer

Quick Start

1. Create a FHIR Resource

2. Serialize to JSON

serialize($patient);

echo $json;
// Output: {"resourceType":"Patient","id":"example-123","active":true, ...}

3. Deserialize from JSON

deserialize($jsonString, PatientResource::class);

echo $patient->id; // "example-123"
echo $patient->active; // true

4. Use FHIRPath

evaluate($patient, 'Patient.name.family');
echo $result[0]; // "Doe"

// Check if patient is active
$result = $evaluator->evaluate($patient, 'Patient.active');
echo $result[0] ? 'true' : 'false'; // "true"
